# Sudoku app

This is a fullstack sudoku Node.js and React.js app.

## [Try it online](https://node-react-sudoku-app.herokuapp.com/)

---

## Installation

```sh
npm install
cd client
npm install
```

## Run in dev mode

```sh
npm start
cd client
npm start
```

## Create client production build

```sh
cd client
npm run build
```
